# Installation
- Python (v2.7)
- word2vec (gensim)
- NLTK (punkt, averaged_perceptron_tagger packages)

# How To Use
1. launch a server
$ python scripts/server.py
2. Open web Firefox browser and access web page
3. Paste Bing news search result with JSON format
4. send your request by pushgin 'Go!' button
